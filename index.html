Demographic Predictor

This project aims to develop a deep learning model to predict the age, gender, and race of individuals based on their facial images. The model is trained on a dataset containing over 20,000 labeled images of faces with corresponding age, gender, and race information.

Github Link: Demographic Predictor Repository

Instagram Unfollow Checker

This project is a JavaScript (JS) tool designed to help users identify Instagram accounts they follow but who do not follow them back. It utilizes the Instagram API to fetch follower and following data and sorts users based on follower count to prioritize those with fewer followers.

Github Link: Instagram Unfollow Checker Repository

Album-Guesser

Album Guesser is a web-based guessing game where players have to guess the correct album name based on a selection of three random songs by a specific artist.
